# chordis
Displays chords entered from the midi keyboard.<br>
MIDIキーボードから入力された和音をディスプレイ左上に表示します。

## Demo
![demo](res/demo.gif)

## 使い方
ローカルPCにPython実行環境を構築し、main.pyを実行します。

### 1. インストール
```
$ python --version
Python 3.10.10

$ pip install -U pip
$ pip install -r requirements.txt
```

### 2. 実行
```
$ pythonw.exe main.py
```
